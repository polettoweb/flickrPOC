# Flickr gallery POC

A proof of concept on querying flickr API and displaying the result on a gallery

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```

### Build Prod Version

```
npm run build
```

### Run functional test (Jest)

```
npm run test
```

### Run e2e test (Cypress)

```
npm run test:e2e
```
